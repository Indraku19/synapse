# Synapse — Marketing & Brand Awareness Strategy

**Tagline:** One agent stores. Every agent learns.  
**Positioning:** Decentralized memory layer for AI agents — infrastructure, not app.

---

## Brand Identity

### Siapa Synapse?

Synapse bukan produk consumer. Synapse adalah **infrastruktur** — seperti AWS, Cloudflare, atau Stripe. Developer membangun di atasnya, bukan end-user yang memakai langsung.

Implikasi marketing:
- Target audience: **AI developer, ML engineer, Web3 builder**
- Bukan: end user umum, non-technical audience
- Tone: teknikal tapi accessible, bukan hype
- Hindari: klaim berlebihan, jargon kosong, janji yang tidak bisa dibuktikan

### Brand Pillars

| Pilar | Artinya |
|---|---|
| **Verifiable** | Setiap knowledge punya bukti on-chain — bukan hanya claim |
| **Composable** | Bekerja dengan agent apapun via REST API atau MCP |
| **Collective** | Satu yang store, semua yang belajar — network effect nyata |
| **Scoped** | Namespace isolation — agent tahu apa yang perlu diketahui, bukan segalanya |

## Channel & Taktik

### 1. GitHub (Prioritas Utama)

Developer menemukan tool baru dari GitHub. README yang bagus = marketing terbaik.

**Checklist README:**
- [ ] Demo GIF di bagian atas (bukan screenshot statis)
- [ ] One-liner yang jelas: *"Decentralized memory layer for AI agents"*
- [ ] Quick start yang bisa dijalankan dalam 3 menit
- [ ] Contoh query yang impressive (namespace isolation demo)
- [ ] Badge: build status, license, on-chain verified

**Target:** 100 GitHub stars dalam 30 hari pertama setelah launch.

---

### 2. X / Twitter

Audience: AI researcher, Web3 developer, crypto builder.

**Content formula yang bekerja:**

*Thread teknikal:*
> "We built a decentralized memory layer for AI agents on @0G_labs.
> Here's why namespace isolation matters and how we implemented it: 🧵"

*Demo clip (paling efektif):*

> - Kiri: agent A store knowledge
> - Kanan: agent B (fresh, 0 knowledge) langsung query dan jawab akurat
> Caption: *"One agent stores. Every agent learns. No web search. No hallucination. Just Synapse."*

*Pain point hook:*
> "Your AI agent knows React, Solidity, AND medicine simultaneously.
> That's not specialized intelligence — that's context contamination.
> Synapse fixes this."

**Hashtags:** #AIAgents #BuildOnWeb3 #0GNetwork #MCP #LLM

**Posting cadence:** 3x/minggu minimum. Konsistensi lebih penting dari frekuensi.

### 3. Developer Communities

**Discord/Telegram 0G:**
- Post demo setiap ada update signifikan
- Jawab pertanyaan developer tentang knowledge sharing
- Jangan spam — contribute dulu, promosi setelahnya

**Discord LangChain / AutoGen / CrewAI:**
- Share use case spesifik: *"How I give my LangChain agent persistent memory across sessions using a decentralized knowledge store"*
- Tutorial, bukan iklan

**Reddit r/MachineLearning, r/LocalLLaMA:**
- Long-form post tentang masalah yang Synapse selesaikan
- Hindari promotional tone — lebih ke "sharing what I built and why"

**HackerNews (Show HN):**
- Format: *"Show HN: Synapse — Decentralized memory layer for AI agents built on 0G blockchain"*
- Timing: Selasa-Kamis, jam 9 pagi EST
- Fokus pada novelty teknikal, bukan fitur
